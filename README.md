# React-Chess
Building Chess with React + Typescript + Sass + Webpack on the OOP basis

To start a development server:
```bash
  npm run dev
```

To launch production: 
```bash
  npm run build
  #then off to the build folder
  cd <build directory>
  #and finally start a server
  npx serve
```
